# Tailwind CSS Design System

The application styling relies on **Tailwind CSS** with custom configuration to establish a dark-themed user interface.

## 🎨 Theme Configuration (`tailwind.config.js`)
* **Typography:** Default typography uses **Outfit** or **Inter** fonts.
* **Colors:**
  * Primary background: Slate-950 (`#020617`).
  * Surface card backgrounds: Zinc-900/Zinc-950 with semi-transparent border gradients.
  * Accent highlights: Emerald (`#10b981`) and Violet (`#8b5cf6`).
* **Interactive Transitions:** Sublte micro-animations are added using hover classes and Tailwind animation utilities.
