# Dependency Map

Key package dependencies.

## ⚙️ Backend Core Dependencies
* **permissionless.js:** Integrates the smart account contracts.
* **viem:** Core utility library for blockchain actions.
* **mongoose:** Connects and queries MongoDB.
* **ioredis:** Redis client supporting rate limiting pipelines.
* **zod:** Validates request payloads.

## ⚙️ Frontend Core Dependencies
* **@tanstack/react-query:** Manages portfolio and transaction caching.
* **@reduxjs/toolkit:** Stores session tokens and theme preferences.
* **lucide-react:** Standard icons set.
