# Configs Manifest

Configuration files map.

## ⚙️ Core Configuration Files
* **`backend/tsconfig.json`:** Configures compiler parameters for TypeScript.
* **`backend/jest.config.js`:** Configures testing environments, loaders, and resets mocks before tests.
* **`frontend/vite.config.ts`:** Configures the Vite asset compiler.
* **`frontend/tailwind.config.js`:** Configures theme variables, typography, and viewport limits.
