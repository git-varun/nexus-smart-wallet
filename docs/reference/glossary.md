# Glossary

Commonly used terms across the project.

## 🔑 ERC-4337 Terms
* **UserOperation:** Structure describing a transaction executed on behalf of a smart account.
* **Bundler:** Node that packages userOperations into standard transactions.
* **Paymaster:** Contract that sponsors gas fees.
* **EntryPoint:** Singleton contract executing smart account operations.

## 💻 Frontend Terms
* **Feature-Sliced Design:** Architectural pattern dividing the frontend into layers, slices, and segments.
* **Viem:** TypeScript interface library for Ethereum.
