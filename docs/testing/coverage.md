# Test Coverage Targets

Coverage metrics and targets.

## 📊 Coverage Thresholds
* **Unit Tests:** Target 90% statement coverage.
* **Integration Tests:** Target 80% database and execution path coverage.
* **Playwright Tests:** Target coverage for core user journeys (signup, smart account prediction, transfers).
