# Playwright Browser Testing

End-to-End browser tests are located in the `frontend/` directory.

## 📁 Playwright Specs
* **auth-verification.spec.ts:** Tests user registration, logins, and session persistence.
* **security-verification.spec.ts:** Tests session key creation, whitelist enforcement, and revocation flows.
* **production-journey.spec.ts:** Verifies the complete user flow, from account prediction to transfers.
